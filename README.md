# Django Article Website

A simple article publishing web application built using Django.

## Features

- Display article listings
- Read full article details
- Dynamic URL routing
- Multiple pages (News, Events, Blogs, About)
- Built using Django templates

## Technologies Used

- Python
- Django
- HTML
- CSS
- SQLite
